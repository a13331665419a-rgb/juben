# micro-drama-writer · 微短剧编剧 Skill

> A professional screenwriting skill for Claude Code, specialized in female-oriented emotional micro-dramas.
> 专为 Claude Code 设计的微短剧编剧 Skill，专注女性情感向短剧创作。

---

## English

### What This Skill Does

`micro-drama-writer` transforms Claude into a professional micro-drama screenwriter with years of experience in female-oriented emotional content. It covers the full production pipeline — from concept and character design to episode breakdowns and final scripts.

**Supported genres:**
- Modern Romance (urban sweet love, flash marriage into wealthy family)
- Revenge & Rebirth (reincarnation, identity reveal, face-slapping)
- Hidden Identity (multiple personas, gradual reveal)
- Historical Fantasy (imperial harem, general & warrior romance)
- Time Travel (modern → ancient, ancient → modern)

### How to Trigger

Just describe what you want to create in natural language:

```
Write a 60-episode micro-drama about a reborn CEO's daughter taking revenge
Design a villain face-slap scene for episode 30
Create character profiles for a flash-marriage drama
Write dialogue for the male lead's confession scene
```

### File Structure

```
micro-drama-writer/
├── SKILL.md                      # Main workflow and quality standards
└── references/
    ├── plot-templates.md         # 8 story frameworks with episode pacing
    ├── character-archetypes.md   # 20+ character archetypes with sample lines
    ├── emotional-hooks.md        # Hook & payoff engineering system
    ├── scene-bridges.md          # 40+ classic scene templates
    └── dialogue-craft.md         # Dialogue principles, golden lines & examples
```

### Installation

Copy the `micro-drama-writer/` folder to your Claude Code skills directory:

- **macOS / Linux:** `~/.claude/skills/`
- **Windows:** `C:\Users\<username>\.claude\skills\`

## 中文说明

### 这个 Skill 是什么

`micro-drama-writer` 让 Claude 化身资深微短剧编剧，专注女性情感向内容创作，覆盖完整编剧流程——从题材策划、角色设计，到分集大纲、完整剧本。

**支持题材：**
- 现代都市：甜宠、闪婚豪门、职场逆袭
- 重生复仇：前世惨死、今世反杀、精准打脸
- 隐藏身份：马甲文、多重身份逐步揭露
- 古装架空：帝宠、将军配、宫斗
- 穿越：现代人穿古代 / 古代人穿现代

### 如何触发

直接用自然语言描述你的需求：

```
帮我写一个重生女总裁复仇的60集短剧
设计第30集的打脸名场面
给一个闪婚剧设计主要角色
写男主表白场景的台词
```
