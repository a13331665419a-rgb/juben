## 1. 都市甜宠：闪婚豪门

**核心公式**：普通女孩 × 顶级豪门 × 契约婚姻 → 真爱

**开场设置**（前3集）：
- 女主因某种意外/误会/家族安排嫁给男主（男主以为是别人 / 女主顶替姐妹 / 协议婚姻）
- 男主对女主漠视甚至厌恶，却给足物质
- 第一集结尾必须有一个"让观众觉得男主会爱上她"的种子

**标准矛盾设置**：
- 白莲花前女友/竹马 → 故意挑拨，制造误解
- 婆婆/小姑子 → 刁难欺负，等待打脸
- 公司危机 → 女主意外解决，男主刮目相看
- 身体接触/危机事件 → 男主开始心动

**爽点节奏**：
```
第1-5集：  入门受气（忍气吞声建立同情）
第6-15集： 偶然崭露头角（小惊喜）
第16-25集：反派加码欺负（积累怒气）
第26-35集：女主大反击 + 男主心动（第一个大爽点）
第36-50集：情感升温 + 虐点插入（前女友/误会）
第51-70集：分离危机（最虐）
第71-90集：男主疯狂追妻（最甜）
第91-100集：反派伏诛 + 官宣爱情
```

**黄金开场台词示范**：
> 男：这门婚事，你只是代替品。
> 女：（淡笑）好。那我就做一个让你忘不掉的代替品。

## 2. 都市逆袭：重生复仇

**核心公式**：被害死的女主重生 → 反杀所有人 → 收获真爱

**开场设置**（前3集必须）：
- 第1集：快速展现前世惨死（被渣男/闺蜜/家人联合害死）
- 第2集：重生回到关键时间节点，女主眼神变了
- 第3集：第一个"重生优势"展现（知道秘密/阻止事件）

**标准复仇对象清单**：
- 渣男（前世爱人，今世踩死）
- 白莲花闺蜜（微笑着补刀）
- 心机继母/姐妹（争家产的）
- 幕后主使大boss（最后揭露）

**真爱男主出场方式**（任选）：
- 前世从未注意过的人，重生后发现他一直在保护自己
- 商业对手，被女主能力折服
- 前世恩人，今世相遇

**爽点节奏**：
```
第1-3集：   前世惨死 → 重生（强烈情绪铺垫）
第4-15集：  精准布局，一个一个坑渣男
第16-30集： 第一个反派彻底社死（第一大爽点）
第31-50集： 主线复仇推进 + 真爱暗线展开
第51-70集： 大boss反扑（最危险时刻）
第71-90集： 终极反转 + 最后一个反派伏诛
第91-100集：真爱修成正果
```

**黄金开场示范**：
> （睁开眼，看着手机屏幕上的日期）
> 女主内心OS：2018年3月15日。距离我死，还有三年。
> （微微一笑，眼神锐利）这一次，死的不会是我。

## 3. 都市隐藏身份：马甲文

**核心公式**：顶级大佬隐藏身份 × 被人轻视/欺负 → 一个接一个打脸

**经典马甲设定**：
- 表面：普通打工人/保洁/护士/助理/小员工
- 实际：集团幕后老板 / 顶级医生 / 国际巨星 / 军方大佬

**马甲揭露节奏**（核心看点）：
- 每隔10集揭一个马甲
- 每次揭露必须配一个反派的极致丑态
- 男主每次目睹都要有反应（震惊 → 重新审视 → 心动）

**反派打脸场景清单**：
- 在公司当众羞辱女主 → 结果女主是董事长
- 嫌弃女主穷酸 → 结果女主买下了她家公司
- 拿着假合同来谈判 → 结果对方CEO是女主
- 在高端场合踩女主 → 主持人叫女主上台领奖

**爽点节奏**：
```
第1-10集：  建立人设落差，第一次被欺负
第11-20集： 第一个马甲揭露（小打脸）
第21-40集： 连续被欺负 + 连续揭马甲（中型打脸）
第41-60集： 最大反派发力 + 最重要马甲揭露（大打脸）
第61-80集： 男主彻底心动追求
第81-100集：终极大boss打脸 + HE
```

## 4. 古装虐恋：帝宠

**核心公式**：深宫女子 × 多疑帝王 → 虐到观众哭 → 帝王后悔求复合

**开场设置**：
- 女主入宫背景（家族 / 和亲 / 阴谋）
- 男主初见：不在意 / 利用 / 误解（种下虐的根）
- 第一集埋下"男主其实注意她"的细节

**古装虐恋标准矛盾**：
- 皇后/贵妃嫉妒陷害
- 帝王多疑，听信谗言
- 家族牵连，被迫割舍
- 身份秘密（女主实为仇人之后）

**哭点设计（最重要）**：
- 帝王当众羞辱女主，女主强撑着不落泪
- 女主为保家人/爱人，独自承受误解
- 帝王下旨赐死后才发现冤枉（必须有求而不得）
- 女主转身离开，帝王第一次失控

**爽点节奏**：
```
第1-10集：  相遇相知，甜蜜短暂
第11-30集： 误解积累，开始虐（轻度）
第31-50集： 帝王疑心，开始打压（中度虐）
第51-65集： 最虐 — 帝王听信谗言，女主至暗时刻
第66-75集： 帝王发现真相，开始后悔
第76-90集： 帝王疯狂补偿，但女主心死（此段最甜也最虐）
第91-100集：女主原谅，皆大欢喜
```

**黄金台词示范**：
> 帝：你为何不求朕？
> 女：（低头）因为妾知道，求了也没用。
> 帝：（愣住）……

## 5. 古装架空：将军配

**核心公式**：独立飒女 × 冷面战神 → 势均力敌的爱情

**特点**：比帝宠少三分怜弱，多三分飒爽，女主绝不软弱

**女主类型**：
- 女将军（能与男主平起平坐打仗）
- 江湖大侠（武功盖世，轻视世俗）
- 谋士/军师（才智出众，布局天下）

**男主类型**：
- 战无不胜的将军，唯独在女主面前吃瘪
- 表面冷漠，私下独宠女主
- 为女主打破一切规则

**爽点设计**：
- 女主在战场上救了男主（反转传统保护关系）
- 男主部下小看女主，被女主当场打脸
- 敌国大将向女主求亲，男主醋意大发
- 朝堂上有人弹劾女主，男主当众护妻

## 6. 穿越：现代人穿古代

**核心公式**：现代知识/能力 × 古代环境 → 开挂逆袭

**穿越后必须快速展现优势**（第2-5集）：
- 医学知识救人（救了男主/皇上）
- 商业头脑（开店铺/搞发明挣大钱）
- 现代常识震惊古人（如何做饭/卫生习惯）
- 预知历史（知道即将发生的大事）

**经典穿越困境**：
- 身份问题（被当作奸细/逃犯）
- 语言/习俗不适应（制造喜剧感）
- 想回现代 vs 舍不得男主（情感冲突）
- 历史节点危机（必须做出选择）

**黄金设定**：穿越到即将发生惨案的节点，女主凭现代知识改变命运

## 7. 穿越：古代人穿现代

**核心公式**：古代贵族/高手 × 现代生活 → 反差萌 + 逐渐适应

**喜剧点设计**：
- 不会用手机/电梯（但学得极快）
- 以为电视里的人是真人（当众拔刀）
- 用古代礼节与现代人交流（被当作怪人）
- 用古代武功处理现代纠纷（当街制服小偷）

**情感线设计**：
- 男主不信他来自古代（误以为神经病）
- 古代人凭借本领在现代立足，男主刮目相看
- 男主发现他的与众不同，开始心动
- 回古代 vs 留现代的抉择

## 8. 通用剧情公式

### 女主崛起三段论
```
第一段：建立弱势形象（让观众心疼）
第二段：触底反弹契机（让观众期待）
第三段：连续爆发反击（让观众爽）
```

### 男主心动五步法
```
第一步：不在意（普通人，可替代）
第二步：好奇（这个女人和别人不一样）
第三步：刮目相看（她比我想象的强）
第四步：不自觉保护（连自己都没意识到）
第五步：承认心动（主动出击）
```

### 反派社死标准流程
```
1. 反派当众嚣张，欺负女主
2. 旁观者反应（观众视角，看热闹）
3. 转折出现（身份揭露/后援到场）
4. 反派表情变化（三连：懵逼→慌乱→崩溃）
5. 女主淡定回应（金句收尾）
6. 旁观者集体哑口无言
```

### 爱情升温关键道具
- 下雨被困同一屋檐（肢体接近）
- 男主为女主挡住危险（保护触发）
- 女主受伤男主亲自处理伤口（温柔触发）
- 深夜只有两人，说了平时不会说的话（心理距离缩近）
- 男主喝醉，说出了真心话（欲盖弥彰）
