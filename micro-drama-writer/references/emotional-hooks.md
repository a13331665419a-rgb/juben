## 一、爽点类型库

### A 类：打脸爽（最通用）

**原理**：先建立落差（反派嚣张），再瞬间反转

**五种打脸方式**：

**① 身份揭露打脸**
```
流程：反派当众羞辱 → 女主淡定接受 → 身份揭露 → 反派社死
关键：揭露时女主必须"刚好"在场，亲眼看见反派的表情
```

**② 实力展现打脸**
```
流程：被质疑能力 → 当场证明 → 所有人哑口无言
关键：展现必须超出所有人预期（不是刚刚够用，是碾压）
```

**③ 预谋布局打脸**
```
流程：反派以为掌控局面 → 女主早已布好局 → 反派入套
关键：要有细节暗示女主早就知道了（回看有彩蛋感）
```

**④ 以彼之道打脸**
```
流程：反派用同样的手段对付女主 → 女主原样奉还加利息
关键：必须更狠、更聪明、更有气势
```

**⑤ 意外帮手打脸**
```
流程：反派以为孤立了女主 → 没想到男主/贵人出现
关键：帮手必须是反派的克星，心理落差最大
```

---

### B 类：宠爱爽（CP 向）

**原理**：男主偏爱女主，其他人都看见了

**五种宠爱方式**：

**① 公开护妻**
> 男主在众人面前，不顾一切站到女主那边

**② 细节关怀**
> 记住女主的细节（喜好/习惯/受过的伤），不声不响地为她准备好

**③ 独占嫉妒**
> 其他男人接近女主，男主的反应（细节越小越甜）

**④ 强势清场**
> 一句话/一个眼神，所有人立刻知道女主受保护

**⑤ 降低姿态**
> 高冷男主为女主破了一次例（第一次笑/第一次道歉/第一次等人）

---

### C 类：逆袭爽

**原理**：从弱势到强势，观众见证全程

**逆袭三要素**：
1. 跌落谷底（够惨，让人心疼）
2. 机遇出现（转折点，合理不突兀）
3. 反击出击（比所有人预期的都厉害）

**逆袭场景设计**：
- 职场逆袭：被当众羞辱 → 当众展示核心能力 → 被领导/客户认可
- 感情逆袭：被渣男嫌弃 → 成为他高攀不起的人 → 淡定拒绝
- 家族逆袭：被家人放弃 → 一己之力撑起门面 → 所有人回来求
- 社会逆袭：被阶层歧视 → 实力碾压 → 让人无话可说

---

### D 类：虐恋爽（哭完还想看）

**原理**：把爱压抑到极限，然后一次爆发

**虐点三层结构**：
1. **外部压力**：反派/家族/误解 强行分开两人
2. **内部压力**：两人都有放不下的牵挂，无法直说
3. **最后一根稻草**：一件事让两人不得不分开

**虐完必甜原则**：
> 虐了几集 = 后续要甜几集（不能虐完就 HE，观众会不满足）

## 二、钩子设计系统

### 开场钩（前10秒）

**目标**：让划走的手停下来

**5种开场钩**：

**① 极致反差开场**
```
画面：豪华别墅/CEO办公室等高端场景
台词："等等，她是谁？"
揭示：打扫卫生的保洁/端盘子的服务员 = 这一切的主人
```

**② 强冲突开场**
```
画面：女主被当众羞辱/逼迫/危险中
台词省略，直接拍脸部特写
目的：立刻引发"她怎么办"的好奇
```

**③ 悬念开场**
```
画面：结局场景（男主跪地/女主离开/爆炸现场）
台词："三天前，我还不知道这一切是怎么发生的。"
倒叙 → 观众会想知道为什么
```

**④ 金句开场**
```
女主：（直视镜头）"如果善良不被善待，那善良就是奢侈品。"
目的：价值观共鸣，女性受众代入感强
```

**⑤ 行动开场**
```
省略铺垫，直接一个震撼性动作（女主当场打人/扔合同/撕婚约）
后续解释原因
```

---

### 集中钩（集中段落保持注意）

每隔 **30-60秒** 需要一个小刺激：
- 一句金句台词
- 一个意外转折
- 一个细节揭示（暗示后续剧情）
- 一次情绪小高潮（小打脸/小温柔/小心动）

---

### 结尾钩（集末必须有）

**目标**：让观众点开下一集

**5种结尾钩**：

**① 高潮截断**
```
最精彩的一刻：女主举起手来……
黑屏
下集预告：……
```

**② 反转揭示**
```
观众以为事情解决了
最后一个镜头：揭示更大的危机/秘密
```

**③ 情感爆发点**
```
男主终于要说出那句话……
女主转身……
黑屏
```

**④ 危机临近**
```
最后一幕：反派的阴谋已经启动，女主还不知道
观众知道，但女主不知道（制造焦虑）
```

**⑤ 意外出现**
```
意想不到的人/物/信息出现
（前女友出现/家人到来/证据曝光）
```

## 三、情绪节奏控制

### 情绪波动曲线（一集内）

```
开场：情绪值 7/10（立刻抓住）
      ↓
展开：情绪值 5/10（稳定叙事，推进情节）
      ↓
中段爽点：情绪值 9/10（小高潮）
      ↓
舒缓：情绪值 6/10（让观众喘口气）
      ↓
结尾钩：情绪值 8/10（带着期待离开）
```

### 整体节奏设计

**虐甜比例**：
- 甜宠剧：虐3 : 甜7
- 虐恋剧：虐6 : 甜4（虐后期有反转大甜）
- 逆袭剧：虐5 : 爽5

**禁止连续出现**：
- 3集以上没有任何爽点（观众会弃剧）
- 3集以上全是甜（观众会觉得无聊）
- 突然BE无任何铺垫（观众会愤怒）

## 四、哭点设计（女性特供）

女性观众哭点 ≠ 悲剧，而是**情感共鸣**

### 最有效的五种哭点

**① 委屈独撑**
> 女主一个人撑着所有压力，还要表现得很好，
> 某一刻终于控制不住哭出来（尤其是在无人看见的地方）

**② 被误解仍不解释**
> 女主知道解释也没用，选择沉默接受
> 观众知道真相，替她心疼

**③ 舍弃所爱**
> 为了保护男主/家人，女主主动离开/放弃
> 越是平静离去，越让人心疼

**④ 迟来的关怀**
> 受了很久的苦，终于有人说了一句：
> "你辛苦了，你不用一个人扛。"

**⑤ 相信终于来了**
> 一直被怀疑的女主，男主终于说：
> "我信你。"

## 五、爽感密度公式

### 黄金密度
```
每30秒：1个小情绪刺激（台词/表情/动作）
每1分钟：1个小爽点
每一集结尾：1个大钩子
每5集：1个阶段性爽点（打脸/表白/揭秘）
每10集：1个重大剧情转折
```

### 检验方法
写完一集后，自问：
1. 前10秒有钩子吗？
2. 中间有1-2个情绪峰值吗？
3. 结尾观众会点下一集吗？
4. 这集有让人截图发给朋友的台词吗？

如果以上有两个回答是"没有"，需要重写。
