# 微短剧编剧工作手册

## 核心创作理念

微短剧的本质是**情绪消费品**：
- 每集 1-3 分钟，必须有情绪峰值
- 女性受众核心诉求：被看见、被珍视、爽感宣泄、情感共鸣
- 节奏法则：**快进矛盾 → 快推高潮 → 强留钩子**
- 爽点密度：每 30 秒一个小爽点，每集结尾一个大钩子

---

## 标准创作工作流

### 第一步：题材定位

确认以下四个维度：

| 维度 | 选项 |
|------|------|
| 时代背景 | 现代都市 / 古装架空 / 穿越 / 年代戏 |
| 核心爽点 | 甜宠 / 虐恋 / 逆袭 / 复仇 / 重生 |
| 女主类型 | 重生马甲 / 飒爽强女 / 傻白甜逆袭 / 隐藏身份 |
| 男主类型 | 霸总深情 / 战神冷面 / 腹黑宠妻 / 先虐后宠 |

→ 题材框架与故事模板：[references/plot-templates.md](references/plot-templates.md)

### 第二步：角色设计

必须完成：
- 女主核心人设 + 隐藏身份/能力（制造反转）
- 男主核心人设 + 情感弱点（制造虐点）
- 反派设定（越蠢越坏越好，衬托女主爽感）
- 关键配角（忠犬闺蜜 / 白莲花姐妹 / 忠实管家）

→ 角色原型完整库：[references/character-archetypes.md](references/character-archetypes.md)

### 第三步：结构大纲

短剧标准节奏（以 100 集为例）：

```
第 1-5 集：    强开场 — 极致反差/冲突，强力钩子
第 6-20 集：   矛盾积累 — 误解/压迫/隐忍，每5集一个爆发点
第 21-40 集：  关系转折 — 男主开始注意/心动，插入虐点
第 41-60 集：  高虐段落 — 误解加深/分离/心碎（女性最爱哭的段落）
第 61-80 集：  反转逆袭 — 身份揭露/能力展现/打脸反派
第 81-95 集：  甜蜜收获 — 男主追妻/宠溺/CP名场面
第 96-100 集： 大结局 — 反派伏诛 + 甜蜜HE
```

集数可按比例缩放（30集短剧 / 60集标准 / 100集长剧）。

### 第四步：爽点钩子设计

每集必须有：
- **开场钩**：前10秒抓住注意力（冲突/反差/悬念）
- **集中爽点**：情绪峰值（打脸/宠爱/逆袭/哭点）
- **结尾钩**：让人忍不住点下一集（悬念截断/反转预告）

→ 爽点类型与钩子设计详解：[references/emotional-hooks.md](references/emotional-hooks.md)

### 第五步：剧本写作

单集剧本格式：
```
【第X集】（集名可选）
[场景：地点 · 时间 · 氛围]

人物名：台词（情绪标注）
[动作/表情描写]

人物名：台词
[转场/画面指示]

---【集末钩子】---
```

→ 台词技巧与金句模板：[references/dialogue-craft.md](references/dialogue-craft.md)
→ 经典桥段参考库：[references/scene-bridges.md](references/scene-bridges.md)

## 质量标准

### 必须做到
- 女主第一集极致反差（表面弱/实则强，或身份悬念）
- 男主对女主态度有清晰转变弧线，且有具体触发点
- 每 5 集设置一个"让观众骂反派"的场景
- 虐点之后必须有甜点补偿（虐得多甜得更多）
- 台词口语化、短促、有力，适合演员发挥

### 绝对禁止
- 女主全程软弱、无反击、无成长
- 剧情平铺直叙，没有爽感节点
- 男主爱意转变没有触发点（突然就爱了）
- 台词过于书面化或冗长解释
- 结局不是 HE（除非明确要求 BE）
